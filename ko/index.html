문제

매일 수많은 PC가 잠금 없이 방치됩니다 — 데이터 유출과 무단 접근의 빌미가 됩니다.

수동 잠금? 잊게 됩니다.

자리를 뜰 때마다 Win+L? 결국 잊습니다. 누구나 그렇습니다.

잠금 해제 = 보안 위협

방치된 PC는 보안 사고의 시작입니다. 데이터 유출, 컴플라이언스 위반, 내부자 위협.

기존 솔루션의 한계

블루투스 페어링은 불안정하고, 화면 타임아웃은 느리고, 스마트카드는 수동 조작이 필요합니다.

작동 원리

세 가지 구성요소, 하나의 매끄러운 경험. 스마트폰이 열쇠입니다.

Step 1

스마트폰

Android 앱이 BLE 신호 송출, 인증, 보안 페어링을 처리합니다.

Step 2

USB 동글

초소형 BLE 동글을 PC USB 포트에 꽂으면, 스마트폰과 Windows를 연결합니다.

Step 3

Windows PC

백그라운드 서비스와 Credential Provider가 OS 수준에서 잠금/해제를 처리합니다.

가까이 자동 해제 | 멀어지면 자동 잠금

사용 시나리오

스마트폰을 들고 다가가서 지문 인증하면 PC가 잠금 해제됩니다. 자리를 뜨면 자동으로 잠금.

기능

보안과 편의성, 타협 없이.

근접 자동 잠금/해제

자리를 비우면 자동 잠금, 돌아오면 자동 해제. 손 하나 안 대고.

생체 인증

스마트폰의 지문 센서를 2차 인증으로 활용. 생체 정보는 기기 밖으로 나가지 않습니다.

원터치 수동 제어

앱에서 한 번 탭으로 잠금/해제. 의도적으로 제어하고 싶을 때.

연결 끊김 시 자동 잠금

BLE 연결이 예기치 않게 끊기면 즉시 잠금. 보안 공백 없음.

Windows 로그인 통합

네이티브 Credential Provider 통합. Windows 잠금 화면과 직접 연동됩니다.

멀티 디바이스 지원

여러 대의 스마트폰을 열쇠로 등록. 하나의 동글에 복수 인증 기기.

아키텍처

세 가지 구성요소가 하나의 시스템으로 동작합니다.

USB 동글

스마트폰과 PC 사이의 다리. USB에 꽂고 BLE로 통신합니다.

  • 전용 MCU
  • Arm TrustZone 보안
  • USB HID 인터페이스
  • BLE 5.2+ 라디오

Windows 소프트웨어

백그라운드 서비스와 Credential Provider가 PC 잠금 상태를 관리합니다.

  • 백그라운드 서비스
  • 네이티브 로그인 모듈
  • 관리 앱
  • Windows 10/11 지원

Android 앱

스마트폰이 곧 열쇠. 기기 관리, 인증, PC 제어를 앱에서.

  • 네이티브 Android
  • BLE 연결
  • 생체 인증
  • Android 12+ 필요

보안 우선

잠금 장치는 가장 약한 고리만큼만 강합니다. ProxiGate는 다층 방어로 설계되었습니다.

BLE 암호화

모든 BLE 통신을 암호화. 커스텀 프로토콜에 CRC16 무결성 검사 적용.

챌린지-리스폰스 인증

모든 잠금 해제 요청에 암호학적 챌린지-리스폰스 교환 필수. 정적 토큰 없음.

기기 바인딩

페어링 시 스마트폰과 동글을 암호학적으로 결합. 미인가 기기는 거부됩니다.

Nonce & 타임스탬프

모든 명령에 nonce와 타임스탬프를 포함하여 재전송 공격을 방지. 각 메시지는 고유합니다.
